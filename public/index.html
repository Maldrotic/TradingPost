Description
Using this site allows users to post musical instruments up for sale to everyone. Users post a picture of their instrument as well as a title, price, and description. Other users can reply to the post with a message. Users can search by keyword and price, as well as filter by instrument, or all at once.

Initial Setup Commands
rake db:setup: This will create the database, load the schema and initialize it with seed data

Start Here
Login with an email and password from the valid users list. After logging in, you will be brought to the posts page. This lists all posts in the database ordered by their created date.

Whats Next
From the posts page, you can click the posts dropdown from the nav bar to go to all posts or your posts. You can also select search posts to be brought to the search view. You can also create a new post. On the right side of the nav is a button to get to your messages, it also shows a count of your unread messages. Further you can view your profile and logout.

Extras
There is no button to reach the categories endpoint, but that is because it is meant to be an administrative view. From here you can add new categories of instruments as well as give a category a new type of instrument.

User Story 1 Validation
As a musician, I would like to be able to search for a specific item for sale so that I can determine if I would like to contact them about an offer.

Steps:
  1. Make sure that database is seeded
  2. Login to an account
  3. Navigate to 'Search Posts'
  4. Select 'Electric guitar' from the instrument list
  5. Click 'Search' button
  6. Verify that posts in the list have the are actual guitars
NOTE: This user story was slightly modified. I ended up having multiple ways to search for a post. Either by entering in a keyword that would match to the post's title, by selecting the type of instrument (in this case, an electric guitar), or by price.

User Story 2 Validation
As a musician, I would like to be able to send an offer to another user so that I can start a chain of communication resulting the purchase or trade for my desired instrument or gear.

Steps:
  1. Login to an account
  2. Navigate to 'All Posts'
  3. Click on desired post (created by another user)
  4. Scroll to bottom and click 'Reply' button
  5. Enter desired message
  6. Click 'Send' button
  7. Logout of account
  8. Login to account that message was sent to
  9. Navigate to messages (by clicking mail icon with 1 unread message badge)
  10. Note that new received message's background has an orange tint
  11. Click message to view contents
  12. Verify that it is the message that was originally sent

User Story 3 Validation
The system takes them through a guided search. The system will ask what kind of instrument they are looking for, then the user will select guitar, or a different instrument. Then the system will ask what kind of guitar they are looking for, or they can bring up the results from there. Depending on the instrument, the system will go through more categories. These categories ultimately will be added by the user so that the system doesn't have to update its own data.

Steps:
  1. Login to an account
  2. Navigate to 'Search Posts'
  3. Select the desired instrument from the instrument select input
  4. Click 'Search'
  5. The resulting list shows all posts of that instrument
NOTE: Also this user story came out a little different. Although it still ends up doing the same thing, it's not necessarily guided. I made it so that from the search field, the user can use any combination of filters (keyword on title, instrument type, minimum price, and maximum price) and depending on what is selected will result in a different list. The search results view will display what parameters are used to filter the posts.
